# LeadMine UI/UX 设计规范

## 1. 品牌识别

### 1.1 Logo设计

**设计概念**：抽象网络节点
- 由5个圆点（节点）和连接线组成的放射状网络图案
- 节点象征"线索点"，连接线象征"数据采集网络"
- 极简黑白风格，现代专业

**Logo变体**：
- 完整版：Logo图标 + "LeadMine"文字
- 简化版：仅Logo图标（用于头像、小尺寸场景）

### 1.4 间距系统

- 基础单位：4px

- 卡片内边距：24px
- 页面边距：24px（移动端16px）
- 列表项间距：12px

### 1.5 圆角

- 按钮/输入框：6px
- 卡片：8px
- 标签/徽章：4px
- 头像：50%

## 2. 组件规范

### 2.1 按钮

**主要按钮**
```css
background: #1A1A1A;
color: #FFFFFF;
border: none;
padding: 10px 20px;
border-radius: 6px;
```

**次要按钮**
```css
background: #FFFFFF;
color: #1A1A1A;
border: 1px solid #E5E5E5;
```

**文字按钮**
```css
background: transparent;
color: #666666;
```

### 2.2 输入框

```css
border: 1px solid #E5E5E5;
border-radius: 6px;
padding: 10px 12px;
background: #FFFFFF;
```

### 2.3 卡片

```css
background: #FFFFFF;
border: 1px solid #E5E5E5;
border-radius: 8px;
padding: 24px;
```

### 2.4 表格

- 表头背景：#F9F9F9
- 表头文字：#666666，12px，加粗
- 行高：52px
- 边框：1px solid #E5E5E5
- 悬停行：#F5F5F5

### 2.5 标签/徽章

```css
padding: 4px 8px;
border-radius: 4px;
font-size: 12px;
font-weight: 500;
```

## 3. 页面布局

### 3.1 整体布局

```
┌────────────────────────────────────────────────────┐
│                    顶部导航栏                       │
│  [Logo]              [搜索] [通知] [用户头像]       │
├──────────┬─────────────────────────────────────────┤
│          │                                         │
│  侧边栏   │              主内容区                   │
│          │                                         │
│ [菜单1]  │                                         │
│ [菜单2]  │                                         │
│ [菜单3]  │                                         │
│          │                                         │
└──────────┴─────────────────────────────────────────┘
```

- 侧边栏宽度：220px（可折叠至64px）
- 顶部导航高度：60px
- 内容区最小宽度：800px

### 3.2 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| 桌面 XL | ≥1440px | 3栏布局 |
| 桌面 | ≥1200px | 标准布局 |
| 笔记本 | ≥992px | 紧凑布局 |
| 平板 | ≥768px | 侧边栏折叠 |
| 手机 | <768px | 全宽单栏 |

## 4. 页面设计

### 4.1 登录页

- 居中卡片布局
- 卡片宽度：400px
- Logo居中上方
- 表单字段：用户名、密码
- "记住我"复选框
- 登录按钮全宽

### 4.2 仪表盘

**顶部统计卡片（4个）**
- 今日新增线索
- 本周新增线索
- 本月新增线索
- 线索总数

**中间区域**
- 左：线索类型分布（饼图/条形图）
- 右：最近线索列表（5条）

**底部**
- 数据源状态列表

### 4.3 线索列表页

**顶部工具栏**
- 搜索框
- 筛选条件（状态、类型、来源、日期）
- 新增按钮（可选）

**数据表格**
- 列：序号、企业名称、事件类型、来源、置信度、状态、发布时间、操作
- 操作：查看、编辑、删除

**底部**
- 分页控件
- 总数显示

### 4.4 线索详情页

**左侧主信息**
- 企业名称（大标题）
- 事件类型标签
- 事件详情
- 涉及金额
- 置信度进度条
- 来源信息
- 状态变更下拉框
- 分配给下拉框
- 销售备注文本域

**右侧来源文章**
- 标题
- 摘要
- 原文链接按钮
- 发布时间

### 4.5 数据源管理页

**卡片网格布局**
- 每个数据源一张卡片
- 显示：图标、名称、类型、状态、最后抓取时间
- 操作：编辑、启用/禁用、手动抓取

**新增按钮**
- 固定在页面右上角

## 5. 动画效果

### 5.1 过渡动画

- 按钮悬停：0.2s ease
- 卡片悬停：0.2s ease，轻微上移
- 页面切换：0.3s ease

### 5.2 加载状态

- 骨架屏（Skeleton）
- 旋转加载图标
- 进度条

---

## 6. 无障碍设计

- 颜色对比度 ≥ 4.5:1
- 支持键盘导航
- 表单标签完整
- 图片 alt 文本
- 聚焦状态可见
